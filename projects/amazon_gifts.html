Amazon Gifts UX Research
  • Heuristic
  • Benchmark
  • Usability Testing
  • Personas
  • UI design

This is a user experience analysis for the current 'gift' feature of the amazon website for desktop.

Heuristic Analysis

This is a first approach to the interface, using the principles of user experience in order to, together with other analysis tools, establish strategies to optimize the platform.

Visibility

The hierarchy of the checkbox for adding a gift receipt makes it hard to find.

The interface does not allow the user to know the difference in the total price when selecting the option to add a gift wrapping.

Tips:

1. Always send a gift receipt when the gift option is selected, as it is very unlikely that the user will not want to include it.

2. Make the total purchase price visible and change it when the wrapping is selected.

System vs. real world

The image displaying the gift does not reflect the physical object.

The gift sending process is not similar to making a physical purchase. The steps are too complex

Tips:

1. Improve the appearance and appeal of the gift wrapping, by including a clear picture of a single object.

2. Display all of the gift options after product selection.

User Control

There are not many options for personalizing the gift.

Tips:

1. When purchasing several products, add the option to send them to different addresses, which will save the user time.

2. If possible, expand the customization possibilities: first, the appearance of the packaging and second The exact delivery date. This will give the user more confidence when using the tool.

Consistency

Making the checkmarks more consistent would make the website more comfortable and easy to learn for users.

Efficiency and Flexibility

It is possible that the user sends gifts to the same person in many different occasions.

Tip:

Add the option of saving the address to the gift sending screen, this employs features alredy integrated on the website.

Benchmark

Goals

This analysis aims to compare Amazon's gift option with those of other competitors, in order to analyze the most significant characteristics of the user experience on those websites.

Target

The user journey will be analyzed from the moment the user has decided to send an item as a gift, to the end of the transaction. It is limited to desktop websites.

Websites analyzed

Three websites were chosen for the analysis: Sephora, as a big shopping company, Uncommon goods, a small company with many gift sending features and The Gift Company, a local business especialized in gifts.

Results

Sephora has more options for card customization, while uncommon goods has more features for sending multiple gifts and choosing the wrapping. The gift company excelled in giving precise options for the user to send the gift, such as exact time and date of shipping.

Conclusions

Priotize improving the visual appeal of using the gift option, either by adding card customization or more attractive images of the wrapping. Do not include options for gift receipts, always send gift receits. Evaluate the viability of shipping gifts on an exact date.

Usability Testing

A live virtual intervew was conducted, where the user displayed their screen and told the interviewer their impressions of the process of using the gift feature on Amazon.

Tasks

1. It's your mother's birthday and you need to buy her a gift. You're on a break from work and you take the opportunity to search on your computer for a good gift. Select a gift for your mother and add it to the basket.

2. You already have your gift selected. Since you don't have a lot of time, you need Amazon to take care of everything. Use the gift feature.

3. Customize all other options for Amazon to deliver your gift.

Summary of user feedback

1. There are too many screens with different steps in order to use the feature, and the order isn't intuitive.

2. The gift wrappings don't seem trustworthy for the cost.

3. The 'gift receipt' option is not very visible and it is slightly confusing.

4. User would like to be able to choose a specific delivery date, so that the gift arrives exactly on the birthday date.

User Interface Design

- All gift options located after proceeding to payment.

- Drop down menu with color vs. grayscale as status feedback for the user.

- Each item has different options.

- All options located in one drop down menu.

- 'Gift receipt' option has been ommited, instead receit is always included.

- Gift wrapping picture is larger and more accurate to real life.

- Amazon 'saved addresses' feature is included.

- The gift wrapping cost has been added, and it is visible from the same screen where the gift options are located.
